Artificial Intelligence

AI is the ability of machines to perform tasks that normally require human intelligence.

Examples:

speech recognition

image detection

chatbots

recommendation systems

Machine Learning

Machine learning allows computers to learn patterns from data.

Example:

Predict house price based on:

size
location
number of rooms
ML Workflow
Data Collection
 ↓
Data Preparation
 ↓
Model Training
 ↓
Model Evaluation
 ↓
Prediction
Types of Machine Learning
Supervised Learning

Data has input + correct output (label).

Example:

email → spam / not spam

Common algorithms:

Linear regression

Decision trees

Unsupervised Learning

No labeled output.

Goal: discover patterns.

Example:

Customer segmentation.

Algorithm example:

K-means clustering

Reinforcement Learning

Agent learns by rewards and penalties.

Example:

robotics

game AI

Computer Vision

Computer vision enables machines to understand images.

Examples:

face detection

object recognition

OCR (extract text from images)

Natural Language Processing (NLP)

Allows machines to understand human language.

Examples:

sentiment analysis

translation

chatbots

Generative AI

Models that generate new content.

Examples:

text generation

image generation

code generation

Powered by Large Language Models (LLMs).
